# Logos de clientes

Solte aqui **um arquivo por cliente**. A seção "Empresas que confiam na Rem Soft"
na home lê esta pasta automaticamente — não é preciso editar código.

## Regras
- Formato: **SVG** (ideal) ou **PNG/WebP com fundo transparente**.
- Nome do arquivo em minúsculas, sem espaços nem acentos.
  Ex.: `cotacasa.svg`, `mv.png`, `unicesusc.webp`.
- A ordem exibida segue a ordem alfabética do nome do arquivo.
  Use um prefixo numérico para controlar a ordem: `01-cotacasa.svg`, `02-iglicho.svg`, ...
- Arquivos que começam com `_` ou `.` são ignorados (útil para guardar a imagem
  composta de origem, ex.: `_origem.png`).
